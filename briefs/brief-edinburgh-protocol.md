# Brief: Edinburgh Protocol Agent

## Objectives
- **Map the Edinburgh Protocol** into Mastra constructs (agents, tools, workflows)
- **Create a Cognitive Anchor**: An agent embodiment of Scottish Enlightenment principles

- **Implement Systems Thinking**: Build tools that analyze incentives, not villains
